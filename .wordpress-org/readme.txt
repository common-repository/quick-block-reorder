Quick block reorder helps to rearrange blocks without opening editor.
 
== Description ==

Often you end up situations which require you to reorder the blocks without using the block editor, then this plugin will
help you to do that easily. Drag and drop your blocks in any order and save it quickly. Rearranging the blocks takes more time
to do it on the editor, but in this plugin, that is made simple.

The blocks are shown with their name and its contents, you can drag & drop them.

 
== Installation ==
 
This section describes how to install the plugin and get it working.
 
1. Activate the plugin

2. Go to manage posts screen, click on the reorder icon

3. A modal will open and you can rearrange the blocks from gutenberg

4. After rearranging click on save, poof! you are done, no need to open editor.

 
== Frequently Asked Questions ==
 
= Can i use it with classic editor? =
 
No, tinymce is not supported by the plugin.
 
= How can i reorder the blocks inside the blocks? =
 
This feature is not supported yet, it will available in the future release.
